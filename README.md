# X-whois
![X-whois preview](X-whois.jpg)

## introduction
X-whois is an internet protocol used to search for information about a domain. X-whois information includes domain name, server name, creation date, expiration date, registrar information.

## Instalations
```
$ pkg update -y && pkg upgrade -y
$ apt update -y && apt upgrade -y
$ pkg install git
$ pkg install python2
$ git clone https://github.com/Whomrx666/X-whois.git
$ cd X-whois
$ python2 X-whois.py
```

## Instructions
- **first**: Install tools according to the instructions above
- **second**: Enter the website you want to target
- **third**: Wait for the process
- **last**: Tools will automatically search information about a domain. X-whois information includes domain name, server name, creation date, expiration date, registrar information

## Note
Enter the target domain without https or http, for example kominfo.go.id
